Zadanie 4 - Teoria JavaScript

1) Kiedy powstał JavaScript (podaj datę)?
JavaScript powstał w 1995 roku.

2) Dlaczego jest to język skryptowy?
JavaScript jest językiem skryptowym, ponieważ jest interpretowany bezpośrednio w przeglądarce, bez potrzeby kompilacji.

3) Gdzie jest wykonywany JS (klient lub serwer)?
JavaScript jest wykonywany po stronie klienta, czyli w przeglądarce.

4) Dlaczego JS to bezpieczne narzędzie?
JavaScript jest bezpieczny, ponieważ działa w sandboxie przeglądarki, co ogranicza dostęp do zasobów systemowych.

5) Czy jest to język obiektowy? Jeśli tak, to dlaczego?
Tak, JavaScript jest językiem obiektowym, ponieważ umożliwia tworzenie i manipulację obiektami.

6) Wymień zalety JS:
Zaleta 1: Szybkie działanie po stronie klienta.
Zaleta 2: Obsługa interakcji z użytkownikiem.
Zaleta 3: Wsparcie dla wielu frameworków.

7) Framework to:
Framework to zbiór gotowych narzędzi i bibliotek wspomagających programowanie aplikacji.

8) Do aplikacji mobilnych można użyć Framework:
Tak, na przykład React Native lub Ionic.

9) Aplikacje responsywne to:
Aplikacje, które dostosowują się do różnych rozmiarów ekranów i urządzeń.
